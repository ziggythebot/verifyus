# Enticeable US Residency Verification Problem

## The Problem

**Client:** Enticeable (recruitment marketing agency)
**Issue:** US recruitment agencies they work with are getting flooded with applications from non-US residents

### Current Pain Points

1. **Foreign applicants** applying with hope of getting visa sponsorship after hire
2. **WeWork verification workaround** — requiring applicants to physically go to WeWork and submit with photo proof
3. **Competitor bots** sending fake applications through the system
4. **System overload** — combination of foreign applicants + bots is "nuking the systems"

## Potential Solution Direction

Use **Web3 KYC/identity verification** tech to automate US residency verification without:
- Manual photo verification
- Physical location requirements
- Privacy invasion
- Bot vulnerability

### Technologies to Explore

1. **ZKPass** — zero-knowledge proof identity verification
2. **Web3 KYC solutions** — decentralized identity verification
3. **Proof of residency** protocols

## Goal

Design an automated, privacy-preserving, bot-resistant US residency verification flow that replaces the manual WeWork workaround.
